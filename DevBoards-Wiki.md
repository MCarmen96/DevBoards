# 🚀 Wiki del Proyecto: DevBoards

## 1. Descripción de la Aplicación
DevBoards es una aplicación tipo pinterest orientada a desarrolladores web tanto para juniors como seniors, que permite coleccionar y visualizar codigo de UI, CSS, HTML,javaScript y TypeScript mediante un sistema de pins y tableros temáticos.

## 2. Roles del Sistema ( separación de roles )

👨‍💻 Dev-Explorer (Visualizador): Usuario interesado en consumir contenido, organizar referencias en tableros y mejorar su productividad técnica.

✍️ Dev-Creator (Creador): Usuario que documenta sus soluciones, comparte fragmentos de código y aporta valor a la comunidad.

## 3.Lanzamientos 

### Release 1 (Estructura Base y Visualización)

- Como usuario(Explorer) quiero poder visualizar pins aleatorios en el index de la app para descubrir nuevo contenido de forma rapida
- Como usuario (Creator) quiero poder subir mis pins para poder compartirlos con el resto de usuarios en mi perfil
- Como usuario(que visualiza contenido) quiero poder guardar pins en mi feed para tener una biblioteca de referencias 
- Como usuario (Explorer) quiero poder hacer guardar y almacenar pins en mi perfil
- Como usuario(Creator) quiero poder añadir una descripción técnica a mis pins para explicar como funciona el código compartido.
 
### Release 2: Interacción y Personalización

- Como usuario (Explorer) quiero crear tableros con nombres especificos para agrupar pins de otros usuarios
- Como usuario quiero poder alternar el tema de la web para mejorar mi comodidad visual y accesibilidad
- Como usuario (Explorer) quiero poder ver una vista de detalle del pisn selecionado para leer su descripcion

### Release 3: Comunidad

- Como usuario(Creator)quiero que en cada pin se muestre el autor de ese pin para informar al usuario que visualiza dicho pin 
- Como usuario(Explorer y Creator) quiero poder dar me gusta a los pins ajenos y comentar en ellos para dar feedback al autor
- Como usuario (Explorer y Creator) quiero poder seguir a otros desarrolladores para que su contenido aparezca en mi feed

### Release 4: Busqueda y Notificaiones

- Como usuario(Explorer) quiero un buscador por palabras claves y etiquetas para filtrar pins por tecnolgias por ejemplo "TypeScript"
- Como usuario quiero recibir notificaciones cuando un usuario al que sigo suba pins o actualize tableros para poder ver el nuevo contenido
