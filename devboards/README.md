# DevBoards App

Aplicación Next.js para desarrolladores - Colecciona y comparte código de UI mediante pins y tableros.

## Características

- Pins con código y visualización
- Likes, comentarios y seguimiento de usuarios
- Búsqueda por palabras clave/etiquetas
- Notificaciones en tiempo real
- 3 temas UX: Usabilidad, No Usabilidad, Accesibilidad

## Inicio Rápido

```bash
npm install
npm run db:push
npm run db:seed
npm run dev
```

## Comandos

| Comando | Descripción |
|---------|-------------|
| `npm run dev` | Desarrollo |
| `npm run build` | Producción |
| `npm run db:push` | Sincronizar BD |
| `npm run db:seed` | Datos de prueba |
| `npm run db:studio` | Prisma Studio |

## Tech Stack

- Next.js 16 (App Router)
- Prisma + SQLite
- NextAuth.js
- Bootstrap 5
